# How To Build LLM Wiki In Obsidian?

This note summarizes a practical walkthrough for building an LLM Wiki in Obsidian as a shared memory layer for humans and agentic AI. Key points:

- Core layers: Raw (sources), Wiki (compiled notes), Schema (rules & templates).
- Tools required: Python, Git, Obsidian, a coding agent, optional local LLM.
- Workflow: capture sources, run ingest script, extract topics/concepts/entities, run maintenance (build, lint, manifest).
- Advanced features: Obsidian firewall to isolate vaults, local model launcher, review/draft workflow for human-in-the-loop.
