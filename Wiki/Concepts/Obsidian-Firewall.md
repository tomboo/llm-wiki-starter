# Obsidian Firewall

An Obsidian-safe wrapper/skill that restricts agent file operations to an approved vault list. Prevents accidental cross-vault writes by validating vault identity before performing CLI actions.

Useful when agents have broad filesystem or Obsidian CLI access.
