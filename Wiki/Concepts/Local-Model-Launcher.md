# Local Model Launcher

Pattern for launching and integrating a local LLM (e.g., Olama):

- Provide one-click commands or launcher scripts to spawn the local model.
- Use a draft/review workflow where model proposals are placed in a review folder before applying changes.
- Automate drafts generation and provide manual approval to avoid low-quality edits.
