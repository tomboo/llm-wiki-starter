# Agentic Skills

Repo-local skills (e.g., `llm-wiki-ingest`, `llm-wiki-maintain`, `llm-wiki-lint`, `llm-wiki-query`) provide repeatable commands agents use to operate the vault safely and deterministically.

Skills wrap scripts and templates so agents don't have to reimplement low-level logic.
