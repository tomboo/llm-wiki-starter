# Memory Layer For Agentic AI

An LLM Wiki acts as a shared memory layer that preserves raw sources and compiles them into structured, linked knowledge that any AI agent (or human) can query. Benefits include traceability, reduced hallucination via knowledge graphs, repeatable ingestion via templates/scripts, and easier integration with local models and agentic workflows.
