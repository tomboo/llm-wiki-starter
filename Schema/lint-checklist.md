# Lint Checklist

Compiled Wiki notes:

- must use one allowed tag: `topic`, `concept`, `entity`, `project`, or `log`
- must include `sources:`
- must keep `source_count` equal to the number of entries in `sources:`
- source links must point to existing files under `Raw/Sources/`

Raw source notes:

- must include `Title`, `Reference`, `Created`, `Processed`, and `tags`
- may not be marked `Processed: true` unless covered by Wiki notes

Source manifest:

- should list each source with `path`, `title`, `processed`, `covered_by`, and `updated`
- should reflect current coverage after build and source-scan updates
